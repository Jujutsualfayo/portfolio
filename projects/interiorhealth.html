Project Summary

A digital health logistics and ordering platform that connects rural patients with essential drugs through health workers and clinics.

Problem Statement

In interior rural regions, patients have limited access to affordable healthcare drugs. This project aims to digitize and streamline the ordering and delivery process, improving access and transparency.

Tech Stack

  • Django REST Framework (Backend API)
  • React + Tailwind CSS (Frontend)
  • PostgreSQL (Database)
  • WebSocket (Real-time updates)
  • Docker (Containerization)
  • M-Pesa API (Payment integration)

Features

  • Role-based authentication (Admin, Health Worker, Patient)
  • Medicine search and ordering
  • Real-time order status tracking
  • Payment integration via M-Pesa
  • Delivery tracking and notifications

Challenges & Solutions

Implementing real-time updates with WebSocket in a scalable way was a major challenge. We overcame this by structuring backend event handling with async workers and ensuring the frontend seamlessly handles connection drops.

Impact

Currently used by Community Health Extension Workers in Makueni County, improving medicine access and order transparency for over 100 users. Plans to scale to other underserved locations are underway.
